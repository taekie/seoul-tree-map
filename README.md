# 서울 가로수 지도

서울시 가로수 257,000그루의 위치를 수종별로 시각화한 인터랙티브 지도.

**[지도 보기 →](https://taekie.github.io/seoul-tree-map/)**

## 기능

- 수종별 필터링 (pill 토글)
- 수목 높이 범위 슬라이더
- 자치구 필터
- 도로명 검색
- 줌아웃: 도로를 따라 선으로 표시 / 줌인: 개별 나무 마커
- 다크 모드

## 데이터

## 기술

- Leaflet.js + CARTO 타일
- 점→선 변환: DBSCAN 공간 클러스터링 + Nearest-Neighbor chain
